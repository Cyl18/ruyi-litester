# Ruyi Lit Tester

Test ruyi with LLVM Integrated tester

Tags: ruyi, lit, schroot

## Dependencies

Framework itself:

+ bash
+ coreutils
+ schroot
+ util-linux
+ yq (depends jq)

Run Lit testcases:

+ LLVM Lit

Run Mugen testcases:

+ grep
+ procps-ng
+ Distro package manager

Refer to configuration files to see testsuite dependencies.
